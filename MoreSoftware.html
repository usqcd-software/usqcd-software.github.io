SUSY LATTICE Authors:  Tom DeGrand and David Schaich
More information:  project website

SUSY LATTICE is parallel code for RHMC simulations of extended-supersymmetric Yang—Mills theories in two and four dimensions, including related measurements. Historically based on the MILC code, this package supersedes the serial C++ code written by Simon Catterall and Anosh Joseph (Comput.Phys.Commun. 183 (2012) 1336, arXiv:1108.1503).
